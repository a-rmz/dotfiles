# dotfiles
or how I set up my development environment

Installs the following programs:
- zsh
- ag
- tmux
- vim (latest, > 8)
- vim-plug
- fzf
- liquidprompt

Also, sets `zsh` as the default shell with a nice-looking base16 colorscheme.
